Class BuiltIns

java.lang.Object
  io.github.JalogTeam.jalog.BuiltIns

public class BuiltIns

Class BuiltIns enumerates all classes that implement built-in predicates. New built-in predicates must be introduced here.

A typical row of the table: 

new ClassInfo("exit", Pred_exit.class, 0, 1),
where
"exit"
Name of a built-in predicate as a string. Must differ from other predicate names in the table.
Pred_exit
Class that implements the predicate. Must always be followed by .class .
0
(optional) The minimum number of arguments of the predicate.
1
(optional) The maximum number of the arguments of the predicate.
If the maximum number of the arguments is omitted the minimum number is the only value.
If both numbers are omitted the predicate accepts any number of arguments.
